内核机制层

RTOS 内核机制

内核模块关注 RTOS 在运行时真正做了什么：它如何选择下一个任务、保存状态、响应中断，以及如何执行决定截止时间是否可信的调度规则。

模块定位

这一层负责把理论解释转化成可执行的内核决策。

  • 解释优先级、就绪队列与派发规则如何相互作用。
  • 说明 TCB 所有权、状态迁移与上下文切换边界。
  • 搭建从时序概念模型到可测量运行时行为的桥梁。

架构视图

中断或事件
-> 任务状态更新
-> 就绪队列变化
-> 调度器仲裁
-> 上下文保存与恢复
-> 下一个任务派发
-> 系统层与实践层观察到时序影响
