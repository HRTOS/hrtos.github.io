内核行为系统

内核行为系统用于分析RTOS在真实运行过程中的调度轨迹、 优先级反转现象以及任务切换的底层执行逻辑，是系统级行为观测核心模块。

内核行为模型定义

Kernel Behavior Model = Scheduling Flow + Priority Inversion + Context Switch Trace + Runtime Execution Path

内核行为入口映射

os_schedule_trigger()

触发内核调度行为入口

os_context_save()

保存当前任务运行状态

os_scheduler_run()

执行调度决策逻辑

os_context_switch()

切换到目标任务上下文

os_context_restore()

恢复任务执行现场
