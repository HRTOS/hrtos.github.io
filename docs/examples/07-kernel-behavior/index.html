内核行为系统

内核行为系统用于分析RTOS真实运行路径，包括调度流程、 优先级反转与上下文切换行为，是系统级执行观测核心。

内核行为模型定义

内核行为（Kernel Behavior） = 调度流程（Schedule Flow） + 上下文切换（Context Switch） + 中断交互（Interrupt Interaction） + 优先级反转（Priority Inversion）
