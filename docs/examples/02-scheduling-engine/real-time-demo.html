实时调度演示（Real Time Demo）

Real Time Demo 用于展示HRTOS中的实时任务执行机制， 包括周期任务、截止时间控制以及确定性调度行为。

机制定义

Real-Time Scheduling = 周期任务触发 + Deadline控制 + 确定性响应

实时行为

实时调度通常包含以下关键机制：

周期任务创建

系统按固定周期触发任务运行。

截止时间约束

任务必须在Deadline前完成执行。

调度决策

调度器根据实时规则选择执行任务。

结果验证

验证任务响应时间是否满足实时要求。

执行流程

1. 初始化任务

创建实时任务并配置周期参数。

2. 启动调度器

进入实时调度模式。

3. 周期触发

系统按Tick触发任务执行。

4. 完成验证

确认任务在规定时间内结束。
