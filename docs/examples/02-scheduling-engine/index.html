Scheduling Core

调度引擎

调度引擎是RTOS内核的核心决策模块，用于在多个就绪任务之间选择 最合适的执行对象，并保证实时性与系统确定性。

调度模型定义

Scheduler = Priority Policy + Run Queue + Context Switch Trigger + Preemption Control
