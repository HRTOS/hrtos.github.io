Resource Management

系统资源

系统资源模块用于描述RTOS对内存、栈空间与运行资源的管理机制， 包括分配策略、碎片化控制与异常保护机制。

资源模型（Resource Model） = 内核内存（Kernel Memory） + 任务栈（Task Stack） + 溢出保护（Overflow Protection） + 分配策略（Allocation Policy）

资源管理模型定义

资源模型（Resource Model） = 内存分配（Memory Allocation） + 栈管理（Stack Management） + 碎片控制（Fragmentation Control） + 溢出保护（Overflow Protection）
