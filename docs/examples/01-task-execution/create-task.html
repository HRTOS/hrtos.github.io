创建任务

创建任务是RTOS调度体系的起点，用于初始化任务控制块、栈空间与调度属性， 并将任务加入就绪队列等待执行。

机制定义

创建任务 = TCB初始化 + 栈分配 + 优先级设置 + 加入就绪队列

核心API： os_task os_task_function os_task_ready os_switch

创建流程

任务控制块初始化

建立任务TCB结构，保存运行状态信息。

栈空间分配

为任务分配独立运行栈。

优先级设置

确定任务调度优先级。

加入就绪队列

任务进入Ready状态等待调度。

调度关联： os_task os_task_state task lifecycle

演示代码


#include <HRTOS.H>
#include <HRTOS_KERNEL.H>

/* 任务ID */
#define TASK_ID  1

/* 任务函数 */
void task_demo(void)
{
    while(1)
    {
        os_nop();
        os_nop();

        os_news_send1(0, 0xA1);

        os_task_service(TASK_ID);
    }
}

/* 创建任务 */
void system_init(void)
{
    os_scheduling(1);

    os_task((unsigned int)task_demo, TASK_ID, 3, 0);

    os_task_release(TASK_ID);
}

系统意义

任务创建决定RTOS调度体系的基础结构，是系统从静态进入动态运行的关键入口。
