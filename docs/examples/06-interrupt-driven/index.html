Interrupt System

中断驱动系统

中断驱动系统用于描述RTOS中断响应机制，包括中断进入流程、 嵌套中断控制与中断延迟分析，是实时系统响应能力的核心组成部分。

中断模型定义

中断模型（Interrupt Model） = 进入机制（Entry） + 优先级嵌套（Priority Nesting） + 上下文保存（Context Save） + 延迟控制（Latency Control）

关键能力

快速响应

ISR进入延迟极低，保证实时响应能力。

嵌套控制

支持高优先级中断抢占低优先级ISR。

上下文保护

完整保存任务执行现场。

实时分析

用于测量系统最坏响应延迟。
