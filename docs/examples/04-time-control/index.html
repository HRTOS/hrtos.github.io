Time System

时间控制

时间控制模块用于描述 RTOS 中的时间管理机制， 包括 Tick 系统（Tick System）、延时机制（Delay）、 软件定时器（Software Timer）、超时控制（Timeout） 与调度时序（Scheduler Timing）。

时间控制模型定义

Time Control Model（时间控制模型） = Tick System（系统节拍） + Delay Mechanism（延时机制） + Timer System（定时器系统） + Scheduler Timing（调度时序）

核心机制说明

Tick 驱动

系统通过周期中断维护全局时间，并驱动调度。

Delay 阻塞

任务进入时间等待队列，由Tick递减唤醒。

Timer 回调

软件定时器基于Tick触发周期或单次回调。

调度联动

时间系统直接影响调度与任务切换时机。
