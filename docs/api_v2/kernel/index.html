内核系统（Kernel）

模块分类：系统核心 / 更新时间：2026-07-03

模块概述

Kernel（内核）是 HRTOS 的核心运行层，负责任务创建、调度管理、 空闲任务执行以及系统底层资源控制。

同时该模块还包含8051平台的内存模型适配接口， 用于实现内部RAM与xdata之间的数据调度与优化访问。

核心功能

  • 任务创建与系统初始化
  • 调度器模式控制
  • 空闲任务与系统钩子
  • 内存填充与基础工具函数
  • 8051 XDATA / DATA 数据映射
  • 内存溢出检测机制
