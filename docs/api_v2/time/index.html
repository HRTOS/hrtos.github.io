时间系统（Time & Tick）

模块分类：系统基础 / 调度时间源 / API v2
⚠ Time模块是 HRTOS 调度节奏的基础，所有任务调度与延时均依赖系统Tick。

模块概述

Time模块提供系统时间基准与延时控制能力， 用于驱动任务调度节奏与实时行为控制。

通过Tick计数器实现统一时间源，并提供毫秒级与秒级时间转换接口。

核心能力

  • 系统Tick计数与配置
  • 任务延时控制（阻塞式调度）
  • 毫秒 / 秒 / Tick 时间转换
  • 系统运行时间统计
