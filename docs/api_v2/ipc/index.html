IPC通信与同步机制

模块分类：系统核心 / 更新时间：2026-07-03

模块概述

IPC（Inter-Process Communication）模块用于实现任务之间的数据传递与同步控制， 是 HRTOS 多任务系统的核心组成部分。

本模块提供信号量、互斥锁、事件、消息队列以及邮箱机制， 支持任务同步与异步通信。

核心功能

  • 任务同步机制（Semaphore / Mutex / Event）
  • 任务间通信机制（Message Queue / Mailbox）
  • 资源共享保护（互斥访问）
  • 事件驱动通知机制
