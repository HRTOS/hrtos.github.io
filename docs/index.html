什么是 HRTOS Documentation Center？

HRTOS Documentation Center 是一个面向实时操作系统（RTOS）与嵌入式系统开发者的技术文档中心， 提供从实时系统基础概念、任务调度模型、中断机制，到内核架构、API接口与实验系统的完整参考资料。

文档体系围绕实时系统设计中的关键问题展开： 时间确定性（Time Determinism）、调度策略、资源约束、中断响应与系统可预测性。

无论是学习 RTOS 基础，还是研究 HRTOS 内核实现， 都可以通过本页面快速进入对应模块。
