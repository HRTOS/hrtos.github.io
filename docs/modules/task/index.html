Task 任务模块

Task模块是HRTOS核心执行体系，负责管理任务创建、任务生命周期、 运行状态、优先级控制以及上下文切换。 它定义了任务作为CPU执行实体的完整模型。

核心内容

Task模块描述任务从建立到退出的完整执行过程， 包括任务管理接口、状态模型以及底层切换机制。

任务体系特点

多级执行模型

普通任务采用0~7优先级， 高速任务使用独立寄存器组， 中断执行层独立管理。

轻量状态设计

通过最小化任务状态模型降低调度开销， 复杂阻塞行为由WAIT机制统一处理。
