IPC通信机制

IPC（Inter-Process Communication）是HRTOS任务间通信与同步的核心模块， 用于解决并发任务之间的状态同步、资源竞争与数据传递问题。

Kernel IPC Task Sync Resource Control Data Transfer

① 状态同步层

用于“条件触发型通信”，特点是轻量、无数据或少数据。

Event（事件）

基于位标志的任务同步机制。
