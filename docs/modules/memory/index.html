Memory Management

内存管理模块定义RTOS中所有内存相关机制， 包括分配、碎片控制、内存保护与栈管理， 为系统运行提供稳定、可控的资源基础。

概述

内存管理是HRTOS内核的重要组成部分，负责控制系统中所有动态与静态内存资源的分配、 访问与保护策略，确保实时系统的确定性与可靠性。
