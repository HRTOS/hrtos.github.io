HRTOS Module

Time 时间管理模块

Time模块提供RTOS时间基准与调度驱动能力， 包括系统节拍、延时机制与定时器系统， 是整个实时系统运行的时间核心。

4 Topics Kernel Time RTOS Core

核心内容

时间模块构成RTOS调度与事件驱动的时间基础。

HRTOS 的 Time 模块不仅提供时间接口，更是整个内核调度的驱动层。 所有任务切换、延时、定时器触发与实时调度，都依赖 Tick 系统推进。

Time 模块内部由 Tick 系统驱动，向下支撑调度与时间控制机制：

Tick System → Scheduler Tick → Delay / Timer → Task Scheduling
