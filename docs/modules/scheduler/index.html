Scheduler 调度器

调度器模块是实时操作系统的核心执行单元， 负责管理任务执行顺序、CPU分配策略与实时性保证机制， 决定系统整体响应能力与确定性行为。

3 Topics Kernel Core RTOS Scheduling

概述

Scheduler负责在多个任务之间进行CPU资源分配， 在不同调度策略下（优先级、实时、轮转）， 确保系统满足实时约束与公平性要求。

核心调度策略

HRTOS提供多种调度模型，用于不同实时场景。

优先级调度

基于任务优先级进行抢占式调度， 保证高优先级任务即时执行。

实时调度

基于时间约束（Deadline）的确定性调度模型， 保证任务按时完成。

时间片轮转
