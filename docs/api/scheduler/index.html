调度管理

模块分类：系统核心 / 更新时间：2026-04-26

模块概述

调度管理是 HRTOS 的核心控制模块，负责系统运行模式切换、调度策略选择以及任务切换执行。 该模块通过时间片与优先级机制，实现多任务的高效调度，是系统实时性的关键保障。

核心功能

  • RTOS运行模式控制
  • 调度策略切换（MYOS / HRTOS）
  • 任务优先级调度
  • 任务上下文切换
  • 调度中断配置
