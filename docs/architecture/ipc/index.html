进程通信（IPC）

IPC（Inter-Process Communication）定义了任务之间的数据交换与同步机制， 是实时系统中任务协作与调度的核心基础。

3 个核心主题 系统架构 任务通信层

模块概述

IPC模块负责解决“任务之间如何协作”的问题，而不是简单的数据传输问题， 其核心目标是实现可预测的同步与调度行为。
