HRTOS 架构总览

内核架构总览（Kernel Architecture）

内核是 HRTOS 的核心控制层，负责调度、上下文切换与中断响应。

核心模块 Kernel 系统控制层

模块概述

Kernel 是 HRTOS 的核心控制层，负责任务调度、上下文切换与中断响应。

内核执行闭环模型（Execution Loop）

内核执行是一个循环结构，由事件触发驱动。

基本路径：

事件触发 → 内核入口 → 调度决策 → 执行任务 → 返回内核
所有系统行为最终都会回到内核调度入口。

内核在系统中的地位

内核是系统中的唯一调度控制中心。

所有任务、中断与通信机制最终都依赖内核进行执行权管理。

内核核心职责

内核职责包括以下内容：

  • 任务调度
  • 上下文切换
  • 中断响应
  • 资源管理
  • 实时调度控制
