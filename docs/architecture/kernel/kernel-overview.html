内核架构（Kernel Architecture）

内核是HRTOS的核心控制层，负责调度执行、任务切换、中断响应与系统资源管理， 决定整个实时系统的运行行为。

核心模块 Kernel 系统控制层

模块概述

Kernel层是所有系统行为的控制中心，它不执行业务逻辑，而是决定执行顺序与资源分配。
