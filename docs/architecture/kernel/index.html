内核架构（Kernel）

内核是HRTOS的执行控制中心，不只是任务管理单元， 而是调度域、时间约束与系统边界的统一控制层， 用于保证系统行为的可预测性与确定性。

模块概述

语义定位

Kernel不是执行器，而是系统确定性的“控制边界维护器”。 它的作用是限制不确定性扩散，而不是增加功能复杂度。
