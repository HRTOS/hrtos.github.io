内核结构（Kernel Structure）

内核结构定义了HRTOS内部的功能分层与职责划分，是理解调度、执行与中断关系的基础模型。

结构概述

内核不是单一模块，而是由多个协同子系统构成，包括调度器、执行控制层、中断处理层与资源管理层。
