HRTOS HRTOS API 手册

系统概览

HRTOS 是一个基于优先级的抢占方式的实时操作系统，支持多任务，同优先级任务采用时间片轮转调度。 系统扩展了中断管理、任务同步、定时器管理、进程通信、文件管理、互斥和内存管理机制等功能。 锁支持互斥体的优先级继承，防止优先级反转问题；创新性突破传统中断处理模式，将处理器支持的32个中断源集中处理， 提高中断可控性并保证任务堆栈安全，同时节约有限硬件堆栈空间。 系统支持动态优先级调整，任务可动态创建、挂起、就绪及删除，并配备内存池管理功能，可动态申请内存空间。
